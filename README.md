# Kilangi - Jewellery Website UI

A modern, responsive UI implementation for a premium jewellery website. Features category filtering and interactive effects built with HTML, CSS, and JavaScript.

## Features

### Design & Layout
- Fully responsive UI optimized for desktop, tablet, and mobile devices
- Clean, elegant aesthetic with cream and green color scheme
- Smooth hover effects and CSS transitions
- Edge-to-edge full-width collection showcase
- Mobile-first approach with dedicated mobile layouts

### Sections

#### Header
- Fixed announcement bar with social media links, search functionality, logo branding, wishlist/account/cart icons, comprehensive navigation menu, mobile-optimized category pills with horizontal scrolling

#### Hero Section
- Dual-image hero layout with overlay content, call-to-action button, responsive grid layout

#### Features Bar
- Product highlights (Anti-Fade Polish, Timeless Design, Lightweight Comfort, Modern Minimal)

#### Bestsellers
- JavaScript-powered category filtering (Rings, Earrings, Pendants, Bracelets, Bangles), product cards with images/pricing/color swatches, CSS hover zoom effects

#### Explore Our Collection
- Desktop: 7-column full-width portrait image grid, mobile: horizontal scrollable grid, category navigation (Rings, Earrings, Pendants, Bracelets, Necklaces, Bangles, Accessories)

#### Recently Viewed
- Horizontal scrollable product carousel, product quick preview

#### Gift Notes Section
- Personalized gift note feature showcase, price-based gift card navigation (Under ₹10,000, ₹20,000, ₹30,000), E-Gift card option

#### Promotional Panels
- Dual promotional banners for seasonal collections, full-width image layouts with overlay text

#### Gifts That Celebrate Every Moment
- Occasion-based gift recommendations (Birthday, Anniversary, Engagement), detailed product cards with descriptions

#### User Gallery
- Mobile-only section for user-generated content, photo upload and gallery viewing options

#### Customer Testimonials
- 4-column testimonial grid (desktop)/horizontal scroll (mobile), star ratings and customer reviews, square testimonial images, text line-clamping for consistent card heights

#### Our Ethos
- Brand story and values section, image and text layout, sustainability messaging

#### Lab-Grown Collection
- Feature showcase for lab-grown gold and silver, icon-based benefits (925 Sterling Silver, Certified Diamonds, Ethical Sourcing, etc.)

#### Promises Section
- Three circular icon features, responsive 2+1 grid layout on mobile

#### Footer
- Brand description and logo, three-column navigation (Our Collection, Quick Links, Policies), rounded copyright section, increased font sizes and bold typography
